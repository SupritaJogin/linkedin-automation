\# LinkedIn Automation – Go (Rod)



⚠️ Educational Proof of Concept Only



This project demonstrates advanced browser automation, stealth techniques, and human-like behavior simulation using Go and the Rod library.



\## Features

\- Human-like mouse movement and scrolling

\- Randomized delays and typing simulation

\- Browser fingerprint masking

\- Session persistence using YAML state

\- Daily rate limiting

\- Modular Go architecture



\## Tech Stack

\- Go

\- Rod (Chrome DevTools)

\- YAML state persistence
